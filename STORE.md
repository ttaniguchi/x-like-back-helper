# X Liked-back Helper (お返し支援ツール)

## ストアの掲載情報

### 説明

```
X（Twitter）で自分のポストに「いいね」をしてくれたユーザーに対して、効率的にお返し（いいね返し）をするための Chrome 拡張機能です。
ページ移動に合わせて自動でターゲットを補佐する「サイドパネル」形式で動作します。
※自動で「いいね」を行うBotではなく、あくまでご自身の手作業（いいね返し）を快適にするための補助ツールです。
※ご自身の「いいね」と思う気持ちを大切にしてください。

【特徴】
・サイドパネル専用設計
拡張機能はブラウザの右側に固定表示されるサイドパネルで動作します。以前のポップアップ版と違い、Xのページを移動しても閉じることがなく、作業をスムーズに継続できます。
・自動リストアップ
「いいね」「リポスト」「引用」などの一覧ページを開くだけで自動的にユーザーを抽出し、専用のリストを作成します。サイドパネルを開いたままでもリアルタイムに追加されます。
・インテリジェント・ナビゲーション
相手のプロフィールページなどで、ポストを一つずつ順番にフォーカスして移動できます。X純正に準拠したショートカット（J/Kキー）で快適に巡回できます。
・クイック「いいね」
フォーカスしている（青い枠で囲まれた）ポストに対して、ショートカット（Lキー）一つで素早く「いいね」を実行できます。
・キーボードショートカットで爆速操作
数字キー (1〜9) でのユーザー選択、Nキーでのメインアクション（相手のホームへ移動など）、Dキーでの完了表示切替、Rキーでのリセット操作など、主要な操作がマウスを使わずキーボードだけで完結します。
・フォロー状況の可視化
相互フォロー (🤝)、フォロワー (💙)、フォロー中 (🔖) を一目で判別できます。
・進捗のリアルタイム表示
残り件数 / 合計件数が表示され、作業の進み具合がひと目でわかります。
・ライト / ダークモード対応
Xの設定に合わせて、またはお好みに合わせてテーマを切り替え可能。目に優しいデザインで作業できます。
・安全設計
自動いいね（Bot）ではなく、ユーザーの操作を補助するツールのため、アカウント停止リスクを最小限に抑えています。
・初めてでも安心のガイダンス
次に何をすればよいか、サイドパネルが常にガイドを表示します。

【使い方】
1. 拡張機能のアイコンをクリック（または Cmd/Ctrl + Shift + X を入力）して、サイドパネルを開きます。
2. X で自分のポストの「いいね」「リポスト」「引用」一覧ページのいずれかを開くと、対象ユーザーが自動的にリストへ追加されます。
3. リスト内の「HOME(N)」をクリック、または『Nキー』を押して、順番にお返ししたい相手のページへ移動します。
4. 相手のプロフィールページ等で『J / Kキー』を使い、お返ししたいポストを特定します。
5. 青いガイド（フォーカス）が表示されているポストに対し、『Lキー』を押して「いいね」を完了させます。
6. 同様にリストの次のユーザーへ進み、全てのお返しが終わったら『Rキー（またはリセットボタン）』でリストをクリアして次の作業へ！
```

## プライバシー

### 単一用途

```
Xで自分のポストにいいねしてくれたユーザーを一覧化し、お返し作業を効率化するためのシンプルな支援ツールです。
```

### 権限が必要な理由

storageが必要な理由

```
取得したユーザーリストを保存し、どのユーザーに「お返し」をしたかの進捗を管理するために使用します。
```

activeTabが必要な理由

```
ユーザーが「リスト取得」ボタンを押した際、現在開いているXの「いいね」一覧ページから、一時的にユーザー一覧を取得するために使用します。
```

sidePanelが必要な理由

```
当拡張機能は、X（Twitter）のページ上でユーザーが「いいね返し」などの作業を連続して行う性質上、常に画面の横にUI（リストや進行状況）を表示し続ける必要があります。 通常のポップアップ（Popup）権限を使用した場合、ユーザーがXの画面をクリックして操作した瞬間に拡張機能のUIが閉じてしまい、抽出したユーザーリストの確認やショートカットツールとしての連続的な作業ができなくなってしまいます。 sidePanel 権限を使用することで、ユーザーがXのページ間を頻繁に遷移（URL移動）してもUIを保持し続けることができ、当拡張機能のコアバリューである「作業状況の可視化とシームレスな操作の補助」を実現しているための必須権限です。
```

tabsが必要な理由

```
ページ遷移のプログラム制御: サイドパネルにリストアップされたユーザーをクリック（または該当する数字キーのショートカットを押下）した際、当該タブをそのユーザーのプロフィールURLへ遷移させるため、chrome.tabs.update() を使用しています。
```

ホスト権限が必要な理由

```
本機能は X (x.com / twitter.com) のページ上でのみ動作し、データを取得するために必要な権限です。
```
