# X Liked-back Helper (お返し支援ツール)

![Icon](icons/icon128.png)

X (Twitter) で自分のポストに「いいね」をしてくれたユーザーに対して、効率的にお返し（いいね返し）をするための Chrome 拡張機能です。

### キーボードショートカット

#### 拡張機能の操作（共通）
- **Command + Shift + X** (Mac) / **Ctrl + Shift + X** (Windows): 拡張機能を開く
  - ※ `chrome://extensions/shortcuts` から変更可能です。

#### ポップアップ開示中の操作
- **数字キー (1〜9)**: リストの上から1番目〜9番目のユーザーへジャンプし、自動的にリストから削除（済みに）します。
- **X**: 一番上のメインアクションを実行します（「いいね一覧へ移動」または「リストの一番上の人へジャンプ」）。
- **D**: 「完了表示」の切り替え
- **R**: リストのリセット

---

## 特徴

- **自動リストアップ**: 「いいね」「リポスト」「引用」などの一覧ページを開くだけで自動的にユーザーを抽出し、専用のリストを作成します。スクロールするだけで次々とリストに追加されていき、ポップアップを開きっぱなしでもリアルタイムに反映されます。
- **フォロー状況の可視化**: 相互フォロー (🤝)、フォロワー (💙)、フォロー中 (🔖) を一目で判別。
- **ライブ進捗表示**: `残り件数 / 合計件数` がリアルタイムに表示されるため、作業の進捗がひと目でわかります。
- **爆速ナビゲーション**: ユーザーカード全体がクリック可能。クリックすると相手のプロフィールへジャンプし、リストから自動でフェードアウトします。
- **フラットデザイン**: 無駄なボタン枠を排除したクリーンな UI で、直感的な操作が可能です。
- **完了フィルタリング**: お返しが済んだユーザーを非表示にし、未完了のタスクに集中できます。
- **マイアカウント連携**: 自分のアイコンから一瞬でプロフィールページに戻れるショートカット機能。
- **安全設計**: 自動いいね（Bot）ではなく、あくまでユーザーの操作を補助するツールのため、アカウント停止リスクを最小限に抑えています。

## インストール方法（開発者モード）

現在、本拡張機能は GitHub から直接インストールして利用いただけます。

1. このリポジトリをクローンまたは ZIP でダウンロードして解凍します。
2. Google Chrome を開き、アドレスバーに `chrome://extensions/` を入力して開きます。
3. 右上の「デベロッパー モード」をオンにします。
4. 「パッケージ化されていない拡張機能を読み込む」ボタンをクリックします。
5. 解凍したフォルダを選択すると、インストールが完了します。

## 使い方

1. X で自分のポストの「いいね」「リポスト」「引用」一覧ページのいずれか（`https://x.com/status/XXXX/likes` など）を開きます。
2. ツイートの詳細ページで、ポップアップ内の「❤️ いいね一覧へ移動」をクリックすると、自動的に対象ページに移動できます。
3. 一覧ページをスクロールすると、対象ユーザーが順次リストへ自動追加されます。
4. リスト内のユーザーカード（「HOME」と表示されている行）をクリックして、順番にお返しを行ってください。
5. 全てのお返しが終わったら「リセット」でリストをクリアして次の作業へ！
