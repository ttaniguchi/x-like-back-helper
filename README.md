# X Liked-back Helper (お返し支援ツール)

![UI Preview](icons/icon128.png)

X (Twitter) で自分のポストに「いいね」をしてくれたユーザーに対して、効率的にお返し（いいね返し）をするための Chrome 拡張機能です。
さらに、自分がフォローしているが相手からフォローされていない「ワンウェイ」ユーザーを特定する監査機能も搭載しています。

### 主な機能

- **「お返し」支援**:
  - **自動リストアップ**: 「いいね」「リポスト」「引用」の一覧ページを開くだけで、ユーザーを自動的に抽出してリスト化します。
  - **クイック・ナビゲーション**: ショートカットキー（J/K）でポストを巡回し、Lキーで一瞬で「いいね」を返せます。
  - **フォロー状況の判別**: 相互フォロー (🤝)、フォロワー (💙)、フォロー中 (🔖) を一目で判別。

- **「ワンウェイ」監査**:
  - **自動ビュー切り替え**: [フォロー中](https://x.com/ttAIart/following) 一覧を開くと、サイドパネルが自動的に「ワンウェイ」リストに切り変わります。
  - **未フォロー者の特定**: 自分がフォローしているが、相手からフォローされていないユーザーだけを抽出してスタックします。
  - **プロフ・キャプション表示**: リスト上で相手の自己紹介文を表示。フォローを整理する際の判断を助けます。

- **安全・安心設計**:
  - **DOM解析のみを使用**: Xの内部APIやGraphQLを直接操作・遮断しないため、アカウントの不審な挙動として検知されるリスクを最小限に抑えています。
  - **ライト / ダークモード対応**: Xのテーマ設定に合わせた見やすいデザイン。

---

### キーボードショートカット (サイドパネル操作時)

- **J**: 次のポストへ移動 (↓) ※X純正準拠
- **K**: 前のポストへ移動 (↑) ※X純正準拠
- **L**: フォーカス中のポストを「いいね」
- **N**: メインアクションの実行
  - ポスト詳細ページでは「いいね一覧へ移動」
  - ユーザーリスト表示中は「リストの一番上のユーザーのページへ移動」
- **R**: リストのリセット（全消去）
- **D**: 「完了済みユーザー」の表示/非表示を切り替え

---

### 使い方

#### 1. いいねのお返し（お返し機能）
1. 自分のポストの「いいね」「リポスト」「引用」一覧を開くと、ユーザーがサイドパネルに自動収集されます。
2. リストのユーザーをクリック、または `N` キーで相手のページへ移動します。
3. `J` / `K` キーでポストを巡回し、青い枠のガイドが表示されたら `L` キーでいいねを返します。

#### 2. フォローの整理（ワンウェイ監査）
1. Xのメニューから自分の「フォロー中」一覧を開きます。
2. サイドパネルが自動的に **「ワンウェイリスト」** に切り替わり、自分をフォローしていないユーザーが表示されます。
3. リストのユーザーをクリックして相手のページへ移動し、フォローの継続を判断できます。

---

### インストール方法（開発用）

1. このリポジトリをダウンロードして解凍します。
2. Google Chrome で `chrome://extensions/` を開きます。
3. 「デベロッパー モード」をオンにします。
4. 「パッケージ化されていない拡張機能を読み込む」からフォルダを選択して読み込みます。
