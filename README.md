# X Liked-back Helper (お返し支援ツール)

![Icon](icons/icon128.png)

X (Twitter) で自分のポストに「いいね」をしてくれたユーザーに対して、効率的にお返し（いいね返し）をするための Chrome 拡張機能です。サイドパネルで動作し、ページ移動に合わせて自動でターゲットを補佐します。

### 主な機能

- **X (旧Twitter) に寄り添ったデザイン**: 公式サイトのトーン（配色、ボタン形状、タイポグラフィ）に合わせ、違和感のないUXを提供します。
- **ライト / ダークモード対応**: Xの設定に合わせて、またはお好みに合わせてテーマを切り替え可能。設定は保存されます。
- **自動リストアップ**: 「いいね」「リポスト」「引用」などの一覧ページを開くだけで自動的にユーザーを抽出し、専用のリストを作成します。
- **インテリジェント・ナビゲーション**: タイムラインやプロフィールページで、ポストを一つずつ順番にフォーカス・移動できます。
- **クイックいいね**: フォーカスしているポストに対して、ショートカット一つでいいねを実行。
- **フォロー状況の可視化**: 相互フォロー (🤝)、フォロワー (💙)、フォロー中 (🔖) を一目で判別。
- **ライブ進捗表示**: `残り件数 / 合計件数` がリアルタイムに表示されます。

---

### キーボードショートカット

#### 共通操作
- **Command + Shift + X** (Mac) / **Ctrl + Shift + X** (Windows): 拡張機能（サイドパネル）を開く
  - ※ `chrome://extensions/shortcuts` から変更可能です。

#### サイドパネル操作時
- **J**: 次のポストへ移動 (↓) ※X純正準拠
- **K**: 前のポストへ移動 (↑) ※X純正準拠
- **L**: フォーカス中のポストを「いいね」
- **N**: メインアクションの実行
  - ポスト詳細ページでは「いいね一覧へ移動」
  - ユーザーリスト表示中は「リストの一番上のユーザーのホームへ移動」
- **数字キー (1〜9)**: リストの該当する番号のユーザーへジャンプし、完了済みにします。
- **D**: 「完了済みユーザー」の表示/非表示を切り替え。
- **R**: リストのリセット（全消去）。

---

### 使い方

1. **サイドパネルを開く**: 拡張機能アイコンをクリックしてサイドパネルを起動します。
2. **ターゲットを収集**: 自分のポストを選んでポストアクティビティ（「いいね」「リポスト」「引用」一覧など）を開くと、ユーザーが自動的にリストに追加されます。
3. **お返しを開始**: リストのユーザー（HOME(N)と表示されているもの）をクリック、または `N` キーで相手のページへ移動します。
4. **ポストを巡回**: 相手のプロフィールページで `J` / `K` キーを使い、お返ししたいポストを特定します。
5. **いいねを実行**: 青い点線の枠（ガイド）が表示されている状態で `L` キーを押すといいねが完了します。
6. **次へ**: 同様にリストの次のユーザーへ進みます。

---

### インストール方法（開発者モード）

1. このリポジトリをクローンまたは ZIP でダウンロードして解凍します。
2. Google Chrome で `chrome://extensions/` を開きます。
3. 「デベロッパー モード」をオンにします。
4. 「パッケージ化されていない拡張機能を読み込む」からフォルダを選択します。
